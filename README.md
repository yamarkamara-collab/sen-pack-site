# sen-pack-site

Site statique "sen pack" hébergé dans ce dépôt GitHub.

Contenu:
- `index.html` : page d'accueil du site
- `senpack_about.html` : page à propos
- `senpack_contact.html` : page de contact
- `senpack_products.html` : page de catalogue
- `senpack_product-1.html` à `senpack_product-4.html` : pages produits
- `styles.css` : styles du site

Pour consulter le site, activez GitHub Pages depuis les paramètres du dépôt et servez le contenu depuis la branche `main` / dossier racine.
